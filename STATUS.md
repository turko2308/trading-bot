## מה הפרויקט
בוט Python (קובץ יחיד `trading_bot.py`) לאיתותי מסחר על זהב (XAU/USD) בטלגרם.
המשתמש מאשר כניסות ידנית (כרגע paper trading — לוחץ "נכנסתי" בלי לפתוח בפלוס500).
עבודה **רק** דרך GitHub web + Render dashboard. אין כלים מקומיים. העלאת קוד: **Add file → Upload files בלבד** (הדבקה נחתכת!).

## תשתית
- Render Background Worker (free), deploy אוטומטי מ-push ל-`turko2308/trading-bot`
- נתונים: Twelve Data API (free, ~800 credits/יום). עובד רק XAU/USD. חסומים: XAG, WTI, מדדים, ETF
- Env ב-Render: `TELEGRAM_TOKEN`, `TELEGRAM_CHAT_ID`, `TWELVE_DATA_API_KEY`, `GIST_ID`, `GIST_TOKEN`, `TZ`
- אחסון קבוע: GitHub Gist (`bot_data.json`) — עובד ומאומת

## ארכיטקטורה (גרסה 3.2) — לשמר בכל שכתוב
1. **לולאה:** כפתורים כל 2 שנ' (getUpdates), סריקה כל 600 שנ'. נתונים חיים בזיכרון; Gist נקרא פעם בהפעלה, נכתב בכל אירוע (+ dirty-retry)
2. **פילטר מגמה קשה:** EMA50 על 1h. מעל +0.3% → רק לונג; מתחת -0.3% → רק שורט; דדזון → כלום. הכיוון נקבע רק כאן
3. **תקרת מתיחה 1.2%:** רחוק מדי מה-EMA → אין איתות
4. **שער ADX:** <20 חוסם; ≥25 מוסיף ניקוד
5. **רצפת סטופ 0.35%** מעל ATR×1.5 (הגבוה מביניהם)
6. **ניקוד:** MACD/פריצה/RSI-תיקון/בולינגר/ADX, מינ' 2 כוכבים + 2 תומכים
7. **Circuit breaker:** 3 הפסדים רצופים ביום → עצירה עד מחר
8. **הגנות:** שבת+ראשון חסומים; נתונים ישנים מ-45 דק' → מדלג
9. **שעון ישראל קשיח** (ZoneInfo) — לא תלוי ב-TZ של השרת
10. **מספור יומי:** `04/07 #2`. **pending בתוך ה-data** (שורד deploy)
11. `/backtest` בטלגרם: סימולציה על 30 יום, 4 ווריאנטים
12. כלכלה: `POSITION_SIZE_OZ=0.75`, USD_ILS=2.99, ספרד 0.77. **המשתמש סוחר 1.5oz — הבוט מציג חצי מהמציאות**
