## מהות הפרויקט
בוט Python — קובץ יחיד `trading_bot.py` (~1,530 שורות) — שולח איתותי מסחר על זהב (XAU/USD) לטלגרם. המשתמש (איזק) מאשר כניסה בלחיצת כפתור; כרגע paper trading. **עדכון חשוב מהמשתמש (10/07): לפעמים הוא נכנס בפועל בזמן האיתות ומאשר בכפתור רק מאוחר יותר** — לכן מחיר האיתות הוא מחיר הכניסה הנכון, וזמן האיתות הוא זמן הכניסה הנכון (לא זמן הלחיצה).
עבודה **רק** דרך GitHub web + Render dashboard (אין כלים מקומיים). העלאת קוד: **Add file → Upload files בלבד**.

## תשתית
- **Render** Background Worker (free tier), deploy אוטומטי מכל push ל-`turko2308/trading-bot`
- **Twelve Data API** (free, ~800 credits/יום): עובד רק XAU/USD. חסומים בחינמי: XAG, WTI, NDX, SPX, ETF
- **Env ב-Render:** `TELEGRAM_TOKEN`, `TELEGRAM_CHAT_ID`, `TWELVE_DATA_API_KEY`, `GIST_ID`, `GIST_TOKEN`, `TZ`
- **אחסון קבוע:** GitHub Gist (`bot_data.json`) — נטען פעם בהפעלה, נכתב בכל אירוע, dirty-retry בכשל

## ארכיטקטורת גרסה 3.2 — לשמר בכל שינוי
1. **לולאה:** כפתורים כל 2 שנ', סריקת שוק כל 600 שנ'. מוניטור פתוחות 24/7; איתותים חדשים 08:00–22:00 ישראל
2. **פילטר מגמה קשה:** EMA50 על 1h (cache 30 דק'). מעל +0.3% → רק לונג; מתחת -0.3% → רק שורט; דדזון ±0.3% → אפס איתותים. **הכיוון נקבע אך ורק כאן**
3. **תקרת מתיחה 1.2%** מה-EMA
4. **שער ADX:** <20 חוסם; ≥25 מוסיף ניקוד
5. **סטופ:** ATR×1.5 עם רצפה 0.35% (בפועל: 10/15 מעסקאות השבוע ישבו על הרצפה — הרצפה היא הסטופ האפקטיבי). טארגט1 = 2× הסטופ (סוגר הכל), טארגט2 מידע
6. **ניקוד 1-5 כוכבים:** MACD / פריצה / RSI טווח תיקון / בולינגר / ADX≥25. שליחה: מינ' 2 כוכבים + 2 תומכים. **ממצא 10/07: על מדגם 3.2 החי הניקוד לא מנבא — ראו "ממצאי הניתוח"**
7. **Circuit breaker:** 3 הפסדים רצופים ביום (לפי close_time) → אפס איתותים עד מחר
8. **הגנות:** שבת+ראשון חסומים; נר ישן מ-45 דק' → מדלג
9. **שעון ישראל קשיח** (ZoneInfo Asia/Jerusalem)
10. **מספור יומי** `10/07 #1`; pending בתוך ה-data; לכל עסקה stars + close_time
11. **`/backtest` בטלגרם — מנוע v2 (הורחב 10/07):** מושך 30 יום (15min+1h), מריץ בסיס + 7 וריאנטים (וריאנט אחד משתנה בכל שורה מול הלוגיקה החיה): דדזון 0.5%/0.6%, לימיט 0.15%/0.25% (פקיעה: שעה=4 נרות, כולל מונה "לא מולאו"), חסימת RSI קיצוני (≤25 חוסם שורט, ≥75 חוסם לונג), שעת כניסה אחרונה 19:00/20:00. הדוח מציג Δ מול הבסיס. השינוי נגע רק ב-`_simulate`/`run_backtest` (שורות ~868–1119); ברירות המחדל משחזרות את הלוגיקה החיה אחד-לאחד (אומת בטסט)
12. **מגבלות:** 2 מקבילות מקס', 10 כניסות/יום, 20 איתותים/יום, cooldown 30 דק', timeout 6 שעות
13. **כלכלה:** `POSITION_SIZE_OZ=0.75`, `USD_ILS=2.99` (שער שוק 10/07: ~3.01 — סטייה 0.7%, סביר; ההצעה החיצונית 3.65 שגויה סופית), ספרד 0.77 נק'. סיכון ~34 ש"ח לעסקה. המשתמש בדמו 1.5oz — מספרי הבוט חצי מהמציאות שלו
